# ✨ TextUtils

TextUtils is a simple and efficient React-based web application that helps users clean, format, and analyze text quickly.

---

## 🚀 Features

- 🔠 Convert text to Uppercase / Lowercase
- ✂️ Remove extra spaces
- 🔢 Count words and characters
- ⏱️ Estimate reading time
- 🧹 Clean and minimal UI
- 🌙 Light/Dark mode (if added)

---

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- CSS / Bootstrap
